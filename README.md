Smart Parking Allocation System 🚗🅿️

A full-stack smart parking management system built using React, Supabase, Tailwind CSS, and Vite.
This project helps manage parking slots, vehicle entries/exits, ticket tracking, and real-time parking availability through a modern dashboard interface.

⸻

✨ Features

* 📊 Live Dashboard Overview
* 🚘 Vehicle Entry Management
* 🅿️ Parking Slot Allocation
* 🎫 Active Parking Ticket Tracking
* 🚪 Vehicle Exit Processing
* 👨‍💼 Admin Panel
* 🌙 Modern Dark UI Design
* ⚡ Real-time database integration with Supabase

⸻

🛠️ Tech Stack

⚙️ Environment Variables

Create a .env file inside the frontend folder:

VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

⸻

🚀 Installation

Clone the repository:

git clone https://github.com/laharisrgupta06/smart-parking-system.git

Move into the project folder:

cd smart-parking-system

Install dependencies:

npm install

Start the development server:

npm run dev
